## 디렉토리 구조

### components 
개별적인 컴포넌트

### functions 
프로젝트 초기화 함수

### layouts
필수적으로 pages에 포함되는 레이아웃들(header, footer)

### menus 
sideBar, navBar 로 구성할 예정 

### pages 
각각의 컴포넌트들과 함수들을 합쳐 보여줄 페이지 

### router
페이지 간의 이동 관리
